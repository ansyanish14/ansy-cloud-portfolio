# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates how to host a static website using Amazon S3.

The website displays a simple HTML page with an image and is publicly accessible through the S3 static website endpoint.

## Architecture
User → Internet → Amazon S3 (Static Website Hosting)

## Steps Performed
- Created S3 bucket
- Disabled Block Public Access
- Added bucket policy to allow public read access
- Enabled static website hosting
- Uploaded index.html and image file
- Verified website using S3 endpoint URL

## Technologies Used
- Amazon S3
- HTML
- AWS IAM Policies
