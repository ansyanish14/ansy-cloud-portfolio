# Project 1: S3 Static Website

This project demonstrates hosting a static website on AWS S3 with public access for web content.

## Project Overview
- Created an S3 bucket to host a static website.
- Uploaded `index.html` and image files.
- Configured bucket policy for public read access.
- Verified website using S3 endpoint URL.

## Implementation Steps
1. Created a globally unique S3 bucket.
2. Disabled Block Public Access.
3. Enabled Static Website Hosting.
4. Uploaded `index.html` and `myphoto.jpg`.
5. Added bucket policy for public read access.
6. Verified website output using S3 URL.
